# DueWise

DueWise is an academic deadline risk and study planning system for students.

The goal of this project is to help students keep track of courses, assignments, quizzes, exams, and projects in one place. Many students have deadlines listed in different places, so this project tries to make the planning process easier.

## Planned Features

1. Add and edit courses
2. Add and edit academic tasks
3. Store due date, estimated work time, difficulty, grade weight, and progress
4. Show upcoming tasks in one dashboard
5. Calculate a risk score for each task
6. Rank tasks by risk level
7. Detect busy days or weeks
8. Suggest what task the student should work on next
9. Show completed and overdue tasks

## Main Idea

DueWise will not only work like a simple to-do list. It will use task information to help decide which tasks are more urgent or risky. For example, a project with high grade weight and low progress may be ranked higher than a small quiz.
